# **Matriz de Protección Proactiva — Plantilla Base (replicable)**

**Chokepoint:** [Nombre]
**Valor del flujo:** % energía/comercio / criticidad industrial
**Dependencias clave:** países/industrias más expuestos

### 1) Vulnerabilidades estructurales

* Geografía (estrechez, densidad de tráfico)
* Dependencia (energía, alimentos, semiconductores)
* Exposición (proximidad a tensiones / clima / congestión)

### 2) Señales tempranas (indicadores)

* Desviaciones de rutas y tiempos de tránsito
* Aumento de primas de seguro y costos de flete
* Incidentes menores recurrentes / cierres parciales
* Cambios regulatorios o avisos de seguridad

### 3) Capacidades actuales

* Monitoreo (AIS, satelital, puertos)
* Presencia de seguridad y coordinación básica
* Procedimientos de navegación y control

### 4) Brechas críticas

* Integración de datos (imagen operativa común)
* Protocolos multinacionales vinculantes
* Redundancia logística/energética
* Velocidad de decisión

### 5) Acciones prioritarias (0–24 meses)

* **0–6 meses:** quick wins (integración de datos, protocolos, simulacros)
* **6–12 meses:** acuerdos operativos y refuerzo de capacidades
* **12–24 meses:** redundancia e infraestructura

### 6) Redundancia

* Rutas alternativas, almacenamiento estratégico, contratos flexibles

### 7) KPIs de resiliencia

* Tiempo de detección / decisión
* % continuidad del flujo
* Volatilidad de precios/seguros
* Tiempo de recuperación

# **Checklist transversal para gobiernos (Anexo E resumido)**

* **Datos:** ¿Existe una imagen operativa común en tiempo real?
* **Umbrales:** ¿Hay niveles de alerta y decisiones preacordadas?
* **Coordinación:** ¿Quién decide y ejecuta en < 6h?
* **Redundancia:** ¿Qué % del flujo puede desviarse hoy?
* **Simulación:** ¿Se ejercita trimestralmente el peor caso?
* **Comunicación:** ¿El mercado recibe señales claras y coherentes?
