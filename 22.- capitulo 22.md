## Capítulo 22

### Crisis simultáneas

Cuando múltiples chokepoints fallan al mismo tiempo

El sistema global no está diseñado para absorber múltiples disrupciones simultáneas.

Está diseñado para optimizar flujos bajo condiciones estables.

Cuando un chokepoint falla, el sistema se adapta.

Desvía rutas.

Ajusta costos.

Reconfigura decisiones.

Pero cuando varios chokepoints fallan al mismo tiempo, esa capacidad se reduce.

Y el sistema entra en una dinámica distinta.

No de adaptación.

Sino de saturación.

Las crisis simultáneas no son la suma de eventos individuales.

Son multiplicadores.

Cada disrupción amplifica a la otra.

Las alternativas que normalmente absorben presión se saturan.

Las rutas secundarias se congestionan.

Los tiempos se extienden.

Los costos se elevan.

El flujo continúa.

Pero bajo condiciones críticas.

El primer efecto es logístico.

Los desvíos se superponen.

Las rutas se alargan.

La capacidad de absorción se reduce.

Los sistemas operan al límite.

El segundo efecto es económico.

Los costos no aumentan de forma lineal.

Se disparan.

Porque la competencia por recursos se intensifica.

Combustible.

Capacidad de transporte.

Infraestructura.

Todo se vuelve más escaso.

El tercer efecto es operativo.

La planificación pierde precisión.

Las decisiones se vuelven reactivas.

Los márgenes de error desaparecen.

El sistema pierde control fino.

El cuarto efecto es financiero.

La incertidumbre aumenta.

Los mercados reaccionan con mayor volatilidad.

Las expectativas se desalinean.

Las decisiones de inversión se retrasan.

El quinto efecto es sistémico.

El sistema deja de comportarse como un conjunto de flujos independientes.

Se comporta como una red interconectada bajo estrés.

Un cambio en un punto afecta múltiples nodos.

Las disrupciones se propagan.

Se retroalimentan.

Se amplifican.

Ese es el punto crítico.

La simultaneidad transforma la naturaleza del riesgo.

Desde una perspectiva estructural, la vulnerabilidad radica en la interdependencia.

Los chokepoints no operan de forma aislada.

Forman parte de un sistema.

Y cuando ese sistema enfrenta múltiples tensiones, su capacidad de respuesta se reduce.

Desde una perspectiva adversaria, este escenario es particularmente relevante.

No es necesario un evento de gran escala en un solo punto.

Múltiples eventos menores en distintos puntos pueden generar un impacto mayor.

Ese es el mecanismo.

Distribuir la presión.

Saturar la capacidad de adaptación.

Forzar al sistema a operar fuera de su equilibrio.

Las capacidades actuales están diseñadas para responder a eventos individuales.

Pero son limitadas frente a crisis simultáneas.

La coordinación se vuelve más compleja.

La velocidad de respuesta se reduce.

Las decisiones se fragmentan.

Una doctrina proactiva en este contexto requiere un enfoque distinto.

Primero, visión sistémica.

Entender las interdependencias.

Segundo, capacidad de coordinación global.

La respuesta no puede ser local.

Tercero, desarrollo de redundancias.

No en un solo punto.

En el sistema completo.

Cuarto, monitoreo de señales múltiples.

No analizar eventos de forma aislada.

Sino en conjunto.

Estas medidas no eliminan el riesgo.

Pero aumentan la capacidad de absorción.

Cuando múltiples chokepoints fallan al mismo tiempo, el sistema no colapsa de forma inmediata.

Pero entra en una fase de estrés.

Se vuelve más lento.

Más costoso.

Más incierto.

Y en ese estado, su estabilidad se reduce.

Comprender las crisis simultáneas es entender el límite del sistema global.

Porque no es la intensidad de un solo evento lo que define el riesgo.

Es la interacción de múltiples eventos.

Y en esa interacción, reside la mayor vulnerabilidad.
