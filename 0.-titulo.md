## 🔧 FORMATO RECOMENDADO EN WORD (KDP)

Para que luzca profesional:

* Título (THE DEVIL’S THROATS / LAS GARGANTAS DEL DIABLO)
  → **Heading 1 o Título**
  → Centrado
  → Mayúsculas
  → Tamaño: 22–26 pt

* Subtítulo
  → **Normal o Heading 2**
  → Centrado
  → Cursiva
  → Tamaño: 14–16 pt

* Espaciado:
  → 2–3 líneas entre título y subtítulo
  → Página limpia (sin headers/footers)
